ABOUT KTOR

Ktor is a framework for building web applications. It means that we can build a web application like a web page or a web api with it. It is based on kotlin and provides asynchronous programming with coroutines.

INSTALLATION

To create your first project you need to install intellij idea. It is a good (and currently the only usable) IDE for kotlin, but it is power hungry. It has a paid version, but for hobby project you can use the free community version with a some limitations.

Although the community version is totally enough for the projects in this tutorial, I should mention that if you are a student you can get a student license for free.

To download the IDE I recommend to use the Toolbox app from this link: JetBrains Toolbox. It is a great tool for installing and managing all your IDE from JetBrains.

Another way is just to download the installer from the links below.
