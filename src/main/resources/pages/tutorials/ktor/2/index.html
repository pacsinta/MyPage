THE BASICS

Ktor is a framework for building web applications. But what ia a web application? What is framework?

A web application means a program in which a client can get data from a server over the network. For example when you visited this website you made a request and my server sent you this webpage.

A framework makes the programming of the server code much easier. It has a lot of features that you can use to build your application. But first we need to understand the communication mechanism between the client and the server.-

There are multiple protocols that are used to communicate between the client and the server. The main ones are HTTP and websockets.

HTTP COMMUNICATION

The communication has two parts. A request and a response. The request is made by the client to the server. And the response is sent by the server to the client.

Http requests has 4 types:

   PUT - used when you want to add data to the server

   GET - used when you want to fetch data from the server

   DELETE - used when you want to delete data from the server

   UPDATE - used when you want to modify data in the server
